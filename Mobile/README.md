# SmartAmenities Android App

Minimal Android frontend scaffold created in the `mobile` branch.

## Configuration

- App name: `SmartAmenities`
- Package: `com.smartamenities.app`
- Language: Kotlin
- Min SDK: 24
- Build scripts: Kotlin DSL (`*.kts`)

## Quick checks

From this directory, you can run:

```powershell
.\gradlew.bat tasks
```

If Android SDK is configured, you can also run:

```powershell
.\gradlew.bat assembleDebug
```
